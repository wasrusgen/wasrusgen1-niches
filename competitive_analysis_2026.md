## Исторические вехи топ-5

- **Битрикс24:** 2003 (1С-Битрикс, Калининград) → 2012 облако → 2025 BitrixGPT 5
- **amoCRM:** 2009 (QSOFT, Токовинин) → 2016 продана 1С → 2025 amoAI

## Незакрытые ниши (стратегические пробелы)

| Проблема | Возможность |
|---|---|
| Мобильное приложение = обрезанная веб-версия | Mobile-first CRM с полным функционалом в смартфоне |
| Нет AI для полевых продаж | Voice-to-CRM: голосовой ввод → автозаполнение карточки |
| Сложный онбординг (недели на настройку) | No-setup CRM: из коробки за 30 минут |
| Нет геолокации для field sales | Карта визитов, оптимизация маршрутов, check-in |
| Нет предиктивного скоринга лидов у МСБ | Прогноз закрытия, ML-скоринг для SMB |

**Главная ниша:** mobile-first B2B CRM + voice-to-CRM + предиктивная аналитика + онбординг за 30 минут.

# 2. КОНСАЛТИНГ — AI-консалтинг для МСБ

## Карта рынка (4 уровня)

```
УРОВЕНЬ 1: Стратегический (Enterprise-only, мин. чек 10+ млн руб.)
Яков и Партнёры │ Б1 (EY Russia) │ Kept (KPMG Russia) │ IBS

УРОВЕНЬ 2: Банковские AI-экосистемы (инструменты, не консалтинг)
Сбер (GigaChat API, СберБизнес Sofт) │ Яндекс (YandexGPT API, Yandex Cloud)

УРОВЕНЬ 3: AI-интеграторы / AI-студии (прямые конкуренты для МСБ)
Insight AI │ NIKTA AI │ Veonix │ IT-Implant │ НейроРешения

УРОВЕНЬ 4: Нишевые / региональные
Просто Хаб (Сибирь) │ Kineiro (SMM) │ Practico.ai
```

## Ценовой анализ

| Формат | Цена | Кто использует |
|---|---|---|
| Hourly rate | 3 500 — 8 000 руб/час | IT-Implant, фрилансеры |
| AI-аудит бизнес-процессов | 150 000 — 400 000 руб. | Insight AI |
| Внедрение чат-бота | 100 000 — 300 000 руб. + 3-25 тыс./мес поддержка | Veonix, NIKTA |
| Автоматизация (RPA) | 500 000 — 1,5 млн руб. | Крупные интеграторы |
| AI-агент под задачу | 500 000 — 2 млн руб. | Enterprise |
| Подписка / сопровождение | 15 000 — 85 000 руб/мес | Kineiro, Сбер |
| Стратегический (Яков, Б1) | от 10 млн руб. | Enterprise |

## Исторические вехи топ-5

- **Яков и Партнёры:** McKinsey 2000 → уход 2022 → выкуп менеджментом → 300+ сотрудников
- **Б1:** Ernst & Young 1989 → уход 2022 → «Б1», 4000+ сотрудников, 9 городов
- **Kept:** KPMG 1990 → уход 2022 → «Kept», только Enterprise
- **Сбер AI:** AI Lab 2019 → GigaChat API 2023 → СберБизнес Sofт 2024-2026
- **Insight AI:** ~2023 → диагностика + BPMN + стратегия + внедрение, МСБ

## Незакрытые ниши

| Пробел | Возможность |
|---|---|
| AI-консалтинг с ROI-гарантией | Модель «платишь с результата» |
| Комплексное сопровождение 6-12 мес. | «AI-директор на аутсорсе» (ретейнер) |
| Понятный оффер без IT-языка | Консалтинг на языке выручки/операционки |
| Региональный рынок не охвачен | Нишевание по географии/отрасли |

**Ключевые барьеры клиентов:** 74% МСБ не имеют AI-стратегии; 47% не знают что автоматизировать; 42% не доверяют AI.

**Главная ниша:** AI-партнёр МСБ, первый шаг до 100 тыс. руб., фиксированный результат, отраслевая экспертиза, язык бизнеса — её на рынке нет.

# 3. РАДАР — мониторинг отзывов Яндекс Карты / 2GIS

**Объём рынка ORM/SERM:** ~50 млрд руб./год (2025), рост с 16 млрд в 2018

|---|---|---|---|---|---|
| **Repometr** | да | да | нет | ~600 руб. | МСБ, сети |
| **Revvy** | да | да | частично | 2 990–4 990 руб. | Локальный бизнес |
| **RocketData** | да | да | **да (ChatGPT)** | индивидуально | Средний/крупный |

## Ценовые зоны

|---|---|---|
| Бюджетный (1 точка) | 500–1 500 руб./мес | Мониторинг, уведомления |

## Критически важно об API

**Официального API отзывов Яндекс Карт НЕ СУЩЕСТВУЕТ.** 2GIS — ограниченный API. Все конкуренты работают через парсинг (серая зона). Это создаёт как риск, так и барьер входа для новых игроков.

## Исторические вехи

- **Brand Analytics:** 2012 → лидер СНГ 2018 → AI-теггинг 2025
- **Repometr:** ~2016 → 42 000+ точек → нет AI-автоответов
- **RocketData:** ~2016 → 30+ геосервисов → AI-автоответы (2024-2025)
- **Revvy:** ~2020 → интеграции с YCLIENTS/iiko → партнёрская программа 2025

## Незакрытые ниши

1. **AI-автоответы + Яндекс/2GIS в доступном сегменте** (до 1 500 руб./точка) — не существует
2. **Прозрачное ценообразование для сетей МСБ (5–50 точек)** — только Repometr публикует цены
3. **Механизм активного управления рейтингом** — мотивация лояльных + фильтрация негатива до публикации
4. **Интеграция с CRM для автоматического триггера** — после оплаты → автоматически просьба об отзыве

**Свободная ниша:** специализированный B2B для сетей 3–30 точек: мониторинг Яндекс+2GIS + AI-автоответы + управление рейтингом + прозрачные тарифы без звонка менеджеру.

## Ключевой стратегический пробел

**ВСЕ российские игроки** работают по схеме:
`парсинг → структурированные данные → дашборд/таблица → клиент сам думает`

**Никто не предоставляет готовые бизнес-выводы.** Competera был ближе всех, но ушёл с рынка РФ.

## Ограничения доступа к платформам

| Платформа | API | Парсинг | Ограничения |
|---|---|---|---|
| **Авито** | Ограниченный | Запрещён | Антибот с 2023, блокировка гарантирована |
| **Ozon** | Seller API | Запрещён | Детектирование VPN с апр. 2026 |
| **Яндекс.Маркет** | Партнёрский | Технически сложен | Высокая ротация структуры страниц |
| **Wildberries** | Для селлеров | Серая зона | Аналитики работают через соглашения |

## Незакрытая ниша

**«Парсинг + готовые выводы для принятия решений»** — свободна на российском рынке.

Что есть: сырые данные в таблицах, дашборды с ценами, алерты.
Чего нет: «вот что означает эта картина рынка», рекомендации по ассортименту, прогноз тренда на языке коммерческого директора.

## AI vs Юрист-человек

| Вариант | Стоимость | Скорость | Охват |
|---|---|---|---|
| Юрист (Москва) | 7 200–24 800 ₽/час | 2–5 дней | Полная ответственность |
| Абонентское юробслуживание | от 30 000 ₽/мес | По запросу | Ограниченный объём |
| AI-анализ (Doczilla, Noroots) | 250–500 ₽/договор | 2–5 минут | Риски + формулировки |
| Яндекс Нейроюрист | ~40–170 ₽/запрос | Мгновенно | 86% точность |

**AI дешевле юриста в 50–100x, но не несёт юридической ответственности — главный барьер доверия.**

## Исторические вехи топ-5

- **Pravo.tech:** 1997 (NAO ПравоТех) → Caselook → экосистема → 1,3 млрд руб. выручки 2023
- **Doczilla:** 2013 → 2018 конструктор → 2019 Pre-seed Sber500 → реестр РФ ПО
- **Яндекс Нейроюрист:** ноябрь 2025 → Yandex B2B Tech → 86% точность
- **GigaLegal (Сбер):** 2024 → GigaChat-based → анализ сделок
- **Noroots:** ~2022-2023 → корпоративная договорная работа → 99% точность (заявлено)

## Незакрытые ниши

1. **Быстрая защита МСБ/B2C от недобросовестных контрагентов** — прямой оффер «договор прислали, что делать» за 5 минут — ПУСТО
2. **Объяснение рисков простым языком** (не юридическим) — никто не делает системно
3. **Мобильный формат / Telegram-бот** — единичные эксперименты
4. **Разовый анализ за 300–500 ₽** без подписки — почти нет
5. **Выявление «серых» схем контрагента** (не только формулировки, но и паттерны: штрафы в 10x, одностороннее изменение условий)

**Свободная ниша:** быстрый (5 мин), доступный (300–500 ₽/договор), понятный (без юридического языка) AI-анализ входящих договоров с акцентом на защиту от недобросовестных контрагентов для ИП, МСБ и физлиц.
